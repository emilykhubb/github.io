# Data Dictionary (Key Fields)

| Field | Definition |
|------|------------|
| property_id | Unique identifier used to track the same property across sales |
| sale_date | Date of transaction |
| price | Sale price |
| postcode | Full postcode (if included) |
| postcode_area | Postcode area (e.g., "SW", "B", "M") |
| resale_within_2y | Flag: resale occurred within 2 years of previous sale |
| resale_within_5y | Flag: resale occurred within 5 years of previous sale |
| first_price | First observed sale price for the property |
| last_price | Last observed sale price for the property |
| pct_change_first_last | Percent change from first to last sale |
