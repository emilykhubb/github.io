# Project Performance & Bonus Calculation Tool (Excel)

## Overview

This project is an Excel-based performance evaluation and bonus calculation tool designed to standardize project-level compensation decisions. The model blends objective metrics with structured leadership evaluations using adjustable weighted scoring.

The workbook separates input fields from calculated outputs to ensure clarity, auditability, and ease of use for leadership teams.

---

## Purpose

- Standardize per-project performance evaluation
- Combine measurable performance data with leadership ratings
- Improve transparency and consistency in bonus decisions
- Enable future filtering and performance trend analysis

## Model Components

### Objective Metrics
- Profit vs target profit
- Efficiency vs target efficiency
- Mistake rate

### Subjective Metrics
- Structured leadership evaluation questions (weighted average)
- Customer satisfaction (if available)

### Final Output
- Composite performance score
- Calculated bonus value
